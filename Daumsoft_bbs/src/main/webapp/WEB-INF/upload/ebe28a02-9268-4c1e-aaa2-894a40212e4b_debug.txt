// 이클립스
1. break 포인트(번호 옆에) 걸기(컨트롤러) 오류모를때는 첫번째에다가 건다.
2. 어디에서 나는지모르면 첫번째 시작 부분에다가 걸기 밑에서도 관리가능
3. 서버에서 Restart in Debug로 시작
4. Debug접속
5. Java(TM) 강종 == task kill
6. 깜빡걸리면 이클립스에 걸린것이다
7. 웹은 멈춰져있음 이클립스에서 F6(다음) F5(상세보기) F8(다음 포인트로이동은 검색 끝나면 자동 종료)
8. 오른쪽마우스 watch누르면 값이 보임
9. watch에서 Expressions탭에서 Add new expression에 값을 입력하면 value가 나옴. Variables는 모든 것들이 나옴 (변수)
10. 구조가 모를 경우 v 꺾새 누르면 볼 수 있음 상세적으로
